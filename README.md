# Tic Tac Toe - React

Este proyecto es una implementación del clásico juego **Tic Tac Toe (Tres en Raya)** desarrollada utilizando **React.js**.
El proyecto fue construido siguiendo como base el tutorial oficial de react y posteriormente se añadieron mejoras visuales.

Tutorial utilizado como referencia:
https://react.dev/learn/tutorial-tic-tac-toe

---

## Características del proyecto

- tablero interactivo 3x3
- Turnos alternados entre jugadores
- Detección automática de ganador
- Detección de empate
- Historial de movimientos
- Posibilidad de regresar a jugadas anteiores
- Botón para reiniciar la partida

---

## Cómo ejecutar el proyecto

1. Clonar el repositorio

git clone https://github.com/tu-usuario/tic-tac-toe-react

2. Entrar a la carpeta del proyecto

cd tic-tac-toe-react

3. Instalar dependencias

npm install

4. Ejecutar el proyecto

npm run dev

5. Abrir en el navegador

http://localhost:5173
